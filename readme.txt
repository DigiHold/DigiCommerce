== Installation ==

= Quick Start Guide =

1. **Upload & Activate**
   * Visit Plugins > Add New
   * Search for "DigiCommerce"
   * Click "Install Now" and then "Activate"
   * Or upload the plugin files manually to `/wp-content/plugins/digicommerce`

2. **Setup Wizard**
   * Follow the setup wizard to configure essential settings
   * Set up your business information

3. **Settings**
   * Click DigiCommerce > Settings
   * Click Payments tab
   * Connect payment gateways (Stripe/PayPal)

4. **Create Your Products**
   * Go to DigiCommerce > Products
   * Add your digital products, services, or courses
   * Upload files, set prices, and configure options

5. **Customize Your Store**
   * Customize the checkout experience
   * Configure email notifications

6. **Launch Your Store**
   * Add products to your pages using blocks
   * Test the complete purchase process
   * Start accepting orders!

= System Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* MySQL 5.6 or higher
* HTTPS secure connection for payment processing

= Recommended Server Configuration =

* PHP Memory Limit: 128 MB or higher
* Max Upload Size: 64 MB or higher (for larger digital products)
* PHP max_execution_time: 60 seconds or higher

== Frequently Asked Questions ==

= What makes DigiCommerce different from other ecommerce plugins? =

DigiCommerce is specifically designed for digital products, services, and online courses, with a laser focus on conversion optimization. Unlike general ecommerce solutions, every feature is tailored to maximize digital sales conversion rates, with specialized workflows for digital delivery, booking systems, and online learning.

= What types of digital products can I sell with the free version? =

The free version supports all types of digital products, including:
- Ebooks and PDFs
- Software and applications
- Digital art and designs
- Music and audio files
- Video content
- Templates and themes
- Downloadable resources

= What additional features do I get with DigiCommerce Pro? =

DigiCommerce Pro unlocks powerful features including:
- Service bookings and appointments
- Online courses and learning management
- Subscription billing
- Software license management
- Affiliate marketing system
- Abandoned cart recovery
- Side cart for improved user experience
- Discount coupons and promotions
- Customer reviews system
- Amazon S3 integration for large files
- Email marketing platform integration

= How secure are the digital downloads? =

DigiCommerce implements industry-standard security for digital products:
- Unique, time-limited download links
- Download attempt limitations
- No direct file path exposure
- Optional Amazon S3 integration for enhanced security

= Which payment gateways are supported? =

The free version includes:
- Stripe (credit cards)
- PayPal

Both gateways support one-time purchases and subscriptions (with Pro).

= How does VAT/tax handling work? =

DigiCommerce includes comprehensive tax management:
- Automatic EU VAT calculation based on customer location
- VAT number validation for B2B sales
- Configurable tax rates by country/region
- Tax-exclusive or tax-inclusive pricing options
- Detailed tax reporting for compliance

= Can I offer services and appointments with DigiCommerce? =

Yes, with the Pro version and Booking addon, you can offer any service-based business:
- Hotel and accommodation bookings
- Restaurant reservations
- Spa and wellness appointments
- Consultation services

- Class registrations
- Any time-based service

= Can I create and sell online courses? =

Absolutely! With the Pro version and Programs addon, you can:

= Is DigiCommerce compatible with my theme? =

DigiCommerce is designed to work with any properly coded WordPress theme. Our templates use clean, standardized code that adapts to your theme's styling while maintaining the conversion-optimized layout.

= Can I customize the checkout process? =

Yes, DigiCommerce offers extensive customization options:
- Multi-step or single-page checkout
- Custom fields for gathering additional information
- Adjustable form fields and layout
- Customizable emails and receipts
- Optional account creation

= Is DigiCommerce suitable for large catalogs? =

Yes, DigiCommerce is optimized for performance even with large product catalogs. The plugin uses efficient database queries, caching mechanisms, and optimized code to ensure fast loading times regardless of catalog size.
